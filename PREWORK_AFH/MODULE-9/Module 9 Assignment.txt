Angular - structural framework for dynamic web apps.  Uses HTML
Vue - progressive frameowrk for building user interfaces.  Vue is designed from the ground up to be incrementally adoptable.
Node - platform build on Chrome's JavaScript runtime for eaily building fast and scalable network applications.  Node. js uses an event-driven, non-blocking I/O model that makes it lightweight and efficient, perfect for data-intensive real-time applications that run across distributed devices.
eXpress - is a Node. js web application server framework, designed for building single-page, multi-page, and hybrid web applications.
Redux - Redux is a predictable state container for JavaScript applications. It helps you write applications that behave consistently, run in different environments (client, server, and native), and are easy to test. Simply put, Redux is a state management tool.
Powershell - PowerShell is the shell framework developed by Microsoft for administration tasks such as configuration management and automation of repetitive jobs. The term 'PowerShell' refers to both – the shell used to execute commands and the scripting language that goes along with the framework.
Python - Python is an object-oriented programming language that provides rapid application development. It was released in 1991 by Guido van Rossum. It has huge demand in in the Rapid Application Development field due to its dynamic binding and dynamic typing options.
Spring Boot - Spring Boot is a lightweight framework that takes most of the work out of configuring Spring-based applications.
Jasmine - Jasmine is an open-source testing framework for JavaScript. It aims to run on any JavaScript-enabled platform, to not intrude on the application nor the IDE, and to have easy-to-read syntax.
API - a set of functions and procedures allowing the creation of applications that access the features or data of an operating system, application, or other service.
